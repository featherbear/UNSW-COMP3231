For setup instructions, look here

Building the Kernel

./rebuild.sh

Running the OS

In ~/cs3231/root…

  • Start - sys161 kernel
  • Wait for debugger before start - sys161 -w kernel

Debugging

In ~/cs3231/root…

  • Execute os161-gdb kernel

  • Use the connect command

  • where -> show current call frames

  • frame n -> Inspect frame n

  • list -> Show source code
